# Network Intrusion Detection — UNSW-NB15

Machine learning classification model to detect malicious network 
traffic using the UNSW-NB15 benchmark dataset.

## What it does
Classifies network traffic as normal or malicious using preprocessing 
pipelines and classification models.

## Tools used
- Python, scikit-learn, pandas
- ColumnTransformer, LabelEncoder for preprocessing
- Exploratory Data Analysis (EDA)

## Dataset
Due to file size limits, only the testing set is included here. 
The full training set is publicly available from the UNSW-NB15 
official dataset source.

## Files
- Assignment2_4.ipynb — main notebook with preprocessing, EDA, and model
- Assignment2_Report_1.docx — written project report
- UNSW_NB15_testing-set.csv — testing data
